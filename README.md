# LanzouGUI-Py
蓝奏云 Python 本地图形化管理工具 | 自动登录 | 可视化文件管理 | 滑动验证码自动处理

![License](https://img.shields.io/badge/许可证-MIT-orange)

## 项目简介
LanzouGUI-Py 是一款基于 Python 开发的**蓝奏云本地桌面工具**，无需浏览器操作，通过轻量化图形界面实现蓝奏云的全自动登录与文件管理，专为日常使用蓝奏云的用户打造，操作简单、本地运行、安全无泄露。

## ✨ 核心功能
- 🚀 **自动登录**：一键输入账号密码，自动处理滑动验证码，后台无感知登录
- 📂 **本地可视化管理**：像操作本地文件夹一样管理蓝奏云文件
- 📁 **文件夹管理**：支持在线创建、删除蓝奏云文件夹
- 📤 **文件上传**：本地文件一键上传至蓝奏云
- 📥 **文件下载**：蓝奏云文件一键下载到本地，支持自定义保存路径
- 🔙 **路径导航**：直观显示当前路径，支持返回上一级目录
- 🔒 **纯本地运行**：所有数据仅存储在本地，无云端上传，保护隐私
- 🎨 **轻量化界面**：简洁无广告，低资源占用，适配 Windows 平台
- 📊 **目录层级支持**：支持根目录外 4 级目录管理（蓝奏云限制）
- 🛡️ **防误操作**：文件夹下载保护、删除确认等安全机制

## 🛠️ 技术栈
- 开发语言：Python 3.8+
- 图形界面：Tkinter（Python 内置，无需额外安装依赖）
- 核心依赖：
  - Playwright（用于自动化浏览器操作，处理滑动验证码）
  - Requests（用于 HTTP 请求）
- 存储：本地配置文件（登录信息加密存储）

## 🚀 快速开始

### 1. 环境要求
- Python 3.8 或更高版本
- Windows 10/11 操作系统
- 蓝奏云账号

### 2. 安装步骤

#### 方法一：源码运行
```bash
# 克隆项目
git clone https://github.com/94kkandkk/LanzouGUI-Py.git
cd 蓝奏云 webdav

# 安装依赖
pip install -r requirements.txt

# 初始化 Playwright
python -m playwright install

# 运行程序
python main.py
```

#### 方法二：使用打包版本
```bash
# 下载最新版本的 release 包
# 解压后直接运行 LanzouGUI.exe
```

### 3. 使用说明

#### 首次登录
1. 运行程序后，在登录界面输入蓝奏云账号和密码
2. 点击"登录"按钮
3. 程序会自动处理滑动验证码（如有）
4. 登录成功后，Cookie 会自动保存到本地

#### 文件管理操作
- **打开文件夹**：双击文件夹即可进入
- **返回上一级**：点击"返回上一级"按钮
- **创建文件夹**：点击"创建文件夹"按钮，输入名称
- **上传文件**：点击"上传文件"按钮，选择本地文件
- **下载文件**：选中文件后点击"下载文件"按钮
- **删除文件/文件夹**：选中后点击"删除"按钮

#### Cookie 管理
- 首次登录后，Cookie 会自动保存到 `lanzou_cookies.txt`
- 下次启动时会自动加载 Cookie，无需重复登录
- 如需切换账号，删除 `lanzou_cookies.txt` 后重新登录

## ⚠️ 注意事项

### 蓝奏云限制
1. **目录层级**：除了根目录外，最多支持 4 级子目录
2. **文件夹下载**：不支持直接下载文件夹，需逐个下载或使用客户端
3. **文件大小**：单个文件上传大小受蓝奏云账号等级限制

### 使用建议
1. 不要频繁上传/删除文件，避免触发蓝奏云风控
2. 定期清理 Cookie 文件，保证账号安全
3. 请勿存储违法违规文件

## 🐛 常见问题

### Q: 登录时提示"密码错误"
A: 请检查账号密码是否正确，注意区分大小写。如果使用了浏览器自动保存的密码，可能需要手动输入。

### Q: 无法创建文件夹
A: 检查是否超过了 4 级子目录限制（根目录 +4 级）

### Q: 上传文件失败
A: 检查文件大小是否超限，网络连接是否正常

### Q: 返回上一级功能异常
A: 当前版本已修复路径导航问题，如仍有问题请提交 issue

### Q: 无法下载文件夹
A: 蓝奏云官方限制，建议进入文件夹后逐个下载文件，或使用蓝奏云客户端打包下载
