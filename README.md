## 项目简介
LanzouGUI-Py 是一款基于 Python 开发的**蓝奏云本地桌面工具**，无需浏览器操作，通过轻量化图形界面实现蓝奏云的全自动登录与文件管理，专为日常使用蓝奏云的用户打造，操作简单、本地运行、安全无泄露。

## ✨ 核心功能
- 🚀 **自动登录**：一键保存登录信息，重启工具自动登录，告别重复输入账号密码
- 📂 **本地可视化管理**：像操作本地文件夹一样管理蓝奏云文件
- 📁 **文件夹管理**：支持在线创建、重命名、删除蓝奏云文件夹
- 📤 **文件上传**：本地文件一键上传至蓝奏云，支持批量/单文件
- 📥 **文件下载**：蓝奏云文件一键下载到本地，支持自定义保存路径
- 💻 **纯本地运行**：所有数据仅存储在本地，无云端上传，保护隐私
- 🎨 **轻量化界面**：简洁无广告，低资源占用，适配全平台Windows

## 🛠️ 技术栈
- 开发语言：Python 3.8+
- 图形界面：Tkinter（Python内置，无需额外安装依赖）
- 核心依赖：蓝奏云非官方API、本地配置文件（登录信息加密存储）

## 🚀 快速使用
1. 克隆本项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 运行主程序：`python main.py`
4. 首次输入蓝奏云账号密码，勾选自动登录，后续一键启动即用

## 📝 使用说明
1. 登录信息**仅保存在本地配置文件**，不会上传至任何服务器
2. 支持普通文件上传/下载，遵循蓝奏云官方文件大小限制
3. 界面操作逻辑与本地文件管理器一致，零学习成本
