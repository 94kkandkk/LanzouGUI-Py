## 项目简介
LanzouGUI-Py 是一款基于 Python 开发的**蓝奏云本地桌面工具**，无需浏览器操作，通过轻量化图形界面实现蓝奏云的全自动登录与文件管理，专为日常使用蓝奏云的用户打造，操作简单、本地运行、安全无泄露。

## ✨ 核心功能
- 🚀 **自动登录**：一键输入账号密码，自动处理滑动验证码，后台无感知登录
- 📂 **本地可视化管理**：像操作本地文件夹一样管理蓝奏云文件
- 📁 **文件夹管理**：支持在线创建、删除蓝奏云文件夹
- 📤 **文件上传**：本地文件一键上传至蓝奏云
- 📥 **文件下载**：蓝奏云文件一键下载到本地，支持自定义保存路径
- � **路径导航**：直观显示当前路径，支持返回上一级目录
- �💻 **纯本地运行**：所有数据仅存储在本地，无云端上传，保护隐私
- 🎨 **轻量化界面**：简洁无广告，低资源占用，适配Windows平台

## 🛠️ 技术栈
- 开发语言：Python 3.8+
- 图形界面：Tkinter（Python内置，无需额外安装依赖）
- 核心依赖：
  - Playwright（用于自动化浏览器操作，处理滑动验证码）
  - Requests（用于HTTP请求）
- 存储：本地配置文件（登录信息加密存储）

## 🚀 快速使用
1. 克隆本项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 初始化Playwright：`python -m playwright install`
4. 运行主程序：`python main.py`
5. 在界面中输入蓝奏云账号密码，点击登录按钮
6. 程序会自动处理登录过程，包括滑动验证码

## 📝 使用说明
1. **登录**：在界面顶部的登录框中输入账号密码，点击登录按钮，程序会在后台自动完成登录过程
2. **文件管理**：
   - 双击文件夹：打开文件夹
   - 点击"返回上一级"：返回上一级目录
   - 点击"创建文件夹"：创建新文件夹
   - 点击"上传文件"：选择本地文件上传
   - 选择文件后点击"下载文件"：下载文件到本地
   - 选择文件或文件夹后点击"删除"：删除选中项
3. **Cookie管理**：登录信息会自动保存到本地的 `lanzou_cookies.txt` 文件中，下次启动时会自动加载
4. **滑动验证码**：程序会自动处理滑动验证码，无需手动操作

## 🎯 技术实现
### 自动登录与滑动验证码处理
- 使用Playwright自动化浏览器操作
- 模拟人类滑动行为，包括加速度、减速度和随机抖动
- 支持多种类型的滑动验证码（腾讯云、网易云等）
- 无头模式运行，后台无感知

### 文件管理
- 动态提取蓝奏云API参数，避免硬编码导致的失效问题
- 支持文件夹层级管理和路径导航
- 自动处理文件密码，无密码文件默认设置为1234
